# 醫療工具集

## 專案介紹
這是一個提供多種醫療評估工具的靜態網頁應用，專為醫療專業人員設計，集成了常用的醫療評估量表和計算工具。所有工具均可離線使用，無需任何後端支持。

## 已實現功能
- **巴氏量表計算器**：評估患者日常生活活動能力(ADL)
- **肌肉力量評估工具**：評估患者上下肢左右側主要關節的肌肉力量
- **改良版艾氏量表評估工具**：評估患者肌肉張力增高或痙攣程度
- **BMI計算器**：計算體重指數並評估健康狀態
- **搜索功能**：快速查找所需工具
- **響應式設計**：適配桌面和移動設備

## 計劃功能
- 藥物劑量計算器
- GCS評分計算器
- 血壓記錄工具
- MMSE評估工具

## 使用方法
1. 打開 `index.html` 瀏覽所有工具
2. 點擊工具卡片上的「立即使用」按鈕進入對應工具頁面
3. 也可使用搜索功能找到所需工具

## 技術實現
- 純HTML、CSS和JavaScript實現
- 無需任何外部庫或框架
- 完全在前端計算，無需傳輸敏感數據
- 優化的移動端體驗

本工具集僅供參考，不能替代專業醫療建議。使用者應在專業醫療人員的指導下使用這些工具的結果。
